# LumenRTC - Friend's Guide

## How to Join the Chat & Call

1.  **Extract**: Right-click `LumenRTC_Setup.zip` and select "Extract All".
2.  **Open Folder**: Open the extracted `LumenRTC` folder.
3.  **Run**: Double-click `LumenRTC.exe`.
3.  **Login**:
    -   Enter your **Name** (e.g., "Friend").
    -   Enter the **Room ID** your friend gave you (e.g., "Room1").
4.  **Chat**: You can type messages and they will appear instantly.
5.  **Video Call**:
    -   Wait for your friend to start the call.
    -   OR type `/call` to start it yourself.
    -   Type `/hangup` to end the video but keep chatting.
    -   Type `/quit` to close everything.

## Troubleshooting
-   If video is black, check if your webcam is used by another app.
-   Ensure you are connected to the internet (or the same network if testing locally).
